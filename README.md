# Js-BallCollide

##Usage
Import to your project

tip: Native javascript does not depend on other libraries

`<script src="js/ballCollide.js"></script>`

`<script>
         window.onload = function() {
             new BallCollide({
                 container: "ballGroup",  // ball container
                 speed: 1                 // ball running spped
             })
         }
     </script>`
